### 조작 방법
1. ⬆와 ⬇ 화살표를 이용하여 속도 조절 가능
2. ⬅와 ➡️ 화살표를 이용하여 방향 전환 가능
3. s키를 누른 상태로 방향 전환 시 덜 꺾임
4. w키를 누른 상태로 방향 전환시 더 꺾임
5. 차량을 완전히 멈추려면 속도가 거의 줄어든 상태서(speed=0.01일때) 스페이스바를 누르면 됨
